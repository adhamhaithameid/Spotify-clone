# SpotifyClone

## About
A high-fidelity Spotify desktop UI clone built with React, Tailwind CSS, and Heroicons. Created as part of the CodeVeda Front-End Engineering internship to showcase responsive design, animated interactions, and utility-first styling.

---

## Features

- Glass-blur and gradient header with focus rings  
- Collapsible left sidebar and hidden right panel on smaller screens  
- Animated card hover states and play-button overlays  
- Sticky footer player with progress, transport controls, volume, device & queue management  
- Light performance optimizations and smooth scrolling
